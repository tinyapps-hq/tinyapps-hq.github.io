# v3 Carnoustie


### 3.1.0
Released Apr-1, 2026

- In Clubs, you can now track head-to-head records between players
- Automatically generate pairings for your matches
- Added an extra confirmation when deleting a paid tournament to prevent accidental deletions
- Fixed a bug that would cause the app to crash when deleting a match in rare cases
- Fixed a bug that would cause the app to become unresponsive when saving a match in rare cases
- Fixed a bug that would allow a participant to edit a tournament in rare cases
- Performance improvements and quality of life changes


### 3.0.0
Released Feb-7, 2026 

- Run a yearly trip or a weekly league? Clubs track records and stats across multiple Two Up tournaments
- With the new Pro membership, you get a full year of ad-free access to Two Up, unlimited Plus tournaments, and the ability to create and manage Clubs
- Added Nassau game type for all tournaments
- Not sure if you want to commit? You can now try Plus for one week free before purchasing a membership
